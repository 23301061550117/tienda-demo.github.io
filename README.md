# Tienda Demo – Proyecto de Simulación de Tienda en Línea

Este proyecto es una simulación de tienda en línea desarrollada con HTML, CSS y JavaScript como parte de la materia de Programación.  
El sistema permite visualizar productos, filtrarlos, buscarlos y agregarlos a un carrito dinámico.



## Características del Proyecto

 Catálogo dinámico de productos  
 Filtro por categorías  
 Buscador de productos 
 Carrito lateral intuitivo  
 Contador de artículos en carrito  
 Subtotal dinámico  


## Extra del Proyecto

### Sistema de Favoritos
Se agregó una funcionalidad adicional para cumplir con el requisito del proyecto: **Favoritos**.

Incluye:
- Ícono para marcar/desmarcar favoritos  
- Persistencia con LocalStorage (los favoritos no se borran al recargar la página)  
- Identificación visual de productos marcados  


## Tecnologías Utilizadas

- **HTML5**
- **CSS**
- **JavaScript**
